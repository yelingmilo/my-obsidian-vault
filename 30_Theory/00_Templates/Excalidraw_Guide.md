# Excalidraw 手写推导配置指南

## 插件安装

在 Obsidian 插件市场中搜索并安装 **Excalidraw** 插件。

## 快速开始

1. 使用命令面板 (`Ctrl+Shift+P`) 输入 `Excalidraw: New Drawing`
2. 或直接创建 `.excalidraw` 后缀的笔记

## 使用场景

### 理论推导
- 手写公式推导过程
- 绘制物理图像和示意图
- 步骤化推导记录

### 实验设计
- 实验装置草图
- 流程图绘制

### 协作
- 导出为 PNG/SVG
- 嵌入到其他笔记

## 常用快捷键

| 快捷键 | 功能 |
|--------|------|
| `E` | 椭圆工具 |
| `R` | 矩形工具 |
| `L` | 直线工具 |
| `P` | 钢笔工具（手写） |
| `T` | 文本工具 |
| `Ctrl+Z` | 撤销 |
| `Ctrl+Shift+Z` | 重做 |

## 搜索手写内容

Excalidraw 支持 OCR 搜索手写文字，可在笔记中通过搜索快速定位。

## 关联笔记

在理论推导笔记底部添加链接：

```markdown
## 手写推导

![[推导过程.excalidraw]]
```

## 导出设置

在 Excalidraw 设置中配置：
- 默认导出格式：PNG
- 包含背景：否
- 缩放：2x
